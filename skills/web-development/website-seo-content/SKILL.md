# Website SEO Content

Use this skill when writing or improving website copy, metadata, conversion
messaging, SEO structure, FAQ, blog outlines, or multilingual versions.

## Output Artifact

Create or update:

```text
docs/content-plan.md
```

## Content Package

For each page include:

- page goal
- target keyword
- SEO title under 60 characters when practical
- meta description around 140-160 characters
- H1
- section headings
- body copy
- CTA copy
- FAQ entries
- internal links
- schema suggestion when useful

## Conversion Rules

- Put the primary CTA above the fold and repeat near the end.
- Use proof: testimonials, logos, certifications, results, case studies.
- Make forms short and clear.
- Match copy to the business maturity: practical for local services, concise for
  SaaS, trust-heavy for finance/health/legal, product-specific for ecommerce.

## Multilingual Support

When requested, create parallel versions for English, Chinese, Spanish, or
Italian. Preserve meaning and conversion intent; do not do literal translation
when local phrasing would convert better.
