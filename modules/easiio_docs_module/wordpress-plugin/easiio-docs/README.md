# Easiio Docs WordPress Plugin

Embeds the reusable Easiio Docs Module in WordPress pages/posts with the `[easiio_docs]` shortcode.

## Shortcode

```text
[easiio_docs site_id="easiio-main" mode="public"]
```

Optional attributes:

```text
api_base="https://chat.easiio.com"
title="Documentation"
subtitle="Guides and manuals"
target_filter="wordpress-shortcode"
status="published"
visibility="public"
require_login="true"
credential_mode="include"
auth_token="[REDACTED]"
```

Admin/editor mode should only be used on protected admin pages:

```text
[easiio_docs site_id="easiio-main" mode="admin" require_login="true"]
```

## Notes

- The plugin renders `docs.js` and `docs.css`; docs content remains in the central Easiio Docs backend by `site_id`.
- Use `require_login="true"` to require a logged-in WordPress user before rendering.
- Keep `auth_token` redacted and only render it on protected pages if using backend bearer-token checks.
- This plugin does not publish content or call WordPress APIs; Phase 4 draft handoff plans are generated by the docs backend and executed separately by Hermes MCP tools after human approval.
